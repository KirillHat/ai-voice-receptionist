# Case Study - Novikov Beverly Hills (Voice Reception)

## Problem

High-value restaurants lose bookings on missed/short calls, especially after hours and during service rush.

## Solution

A 24/7 AI phone receptionist that:
- answers instantly
- speaks naturally
- qualifies and structures reservation demand
- pushes actionable data to operations tools

## Why this project is strong for portfolio

- Real telephony workflow (not just chat)
- Voice + data + operations integration
- Practical hospitality use case with measurable business impact

## KPIs to track

- Answer rate
- Qualified call rate
- Booking conversion rate
- Average handling time
- Missed-call recovery rate
- % calls with successful recording and QA review
